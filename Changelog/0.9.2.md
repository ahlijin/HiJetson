# Changelog / 更新日志

## 0.9.2 (2026-06-03)

### 新增
- **唤醒词检测节点 (voice_wake_word)**
  - 基于 OpenWakeWord 的流式唤醒词检测
  - 预置模型: hey_jarvis / alexa / hey_mycroft 可选
  - 通过 `voice_params.yaml` 配置模型和检测阈值
- **ASR 唤醒门控**
  - ASR 节点新增 `/voice/wake_word` 订阅
  - 唤醒后 5s 窗口内才处理 VAD 语音片段，其余时间忽略
  - 支持禁用唤醒词 (`wake_word_enabled: False`) 恢复直通模式
- 新 Topic `/voice/wake_word` (std_msgs/String)：唤醒词触发信号

### 配置
- `voice_params.yaml` 新增 `voice_wake_word` 参数段
  - `model_name`: 唤醒词模型（默认 `hey_jarvis`）
  - `threshold`: 检测阈值（默认 0.5）
  - `cooldown`: 防重复触发间隔（默认 3.0s）
- `voice_params.yaml` 新增 `voice_asr.wake_word_*` 参数
  - `wake_word_enabled`: 是否启用唤醒门控（默认 True）
  - `wake_word_timeout`: 唤醒后监听窗口（默认 5.0s）
  - 关闭唤醒门控: `wake_word_enabled: False`（恢复 v0.2 行为）
