# Changelog / 更新日志

## 0.9.1 (2026-06-02)

### 修复
- **PulseAudio 冲突导致 input overflow**
  - `scripts/run_voice.sh`: 添加 pasuspender 包装，临时暂停 PulseAudio 让出 ALSA 设备，退出后自动恢复
  - Capture 节点 device_index=0 直连 ALSA hw:0,0
- **faster-whisper 在 aarch64 无 CUDA wheel**
  - 后端切换为 openai-whisper (PyTorch CUDA)
  - `package.xml`: 移除 faster_whisper 依赖
  - `setup.py`: 更新描述
- **VAD 环境噪音误触发**
  - VAD mode 1→3（最严格）
  - `voice_vad_node.py`: 新增 min_energy RMS 能量门限参数

### 增强
- **ASR 模型 tiny→base**: 37M→72M 参数，中文识别准确率显著提升
- **高通滤波**: Capture 节点加入 300Hz Butterworth 4阶 HPF，去除 Jetson 139Hz 风扇噪声
- **帧对齐**: Capture frame_size 1600→1920 (120ms = VAD 30ms × 4)，消除每帧约 10% 数据浪费
- **单声道直采**: channels 2→1，省去混音步骤

### 配置
- **参数中心化**: Launch 文件取消硬编码，改为从 `src/config/voice_params.yaml` 加载
  - 文件格式增加 ROS2 标准的 `ros__parameters:` 层级
  - 修改参数只需编辑 YAML，无需改 launch 文件

### 文件变更
| 文件 | 变更 |
|------|------|
| scripts/run_voice.sh | +pasuspender, 更新描述 |
| src/launch/hijetson_voice.launch.py | 重写: 从 YAML 加载参数 |

### 验证
- Capture 稳定 8.33Hz (120ms), 无 overflow
- VAD voice_activity=false, 环境噪音不触发
- Whisper base 推理: 5s 音频 → 1.3s (CUDA)
- 空闲显存: 4.0GB / 7.4GB
