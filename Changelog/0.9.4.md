# Changelog / 更新日志

## 0.9.4 (2026-06-29)

### 新增
- **合并 `lite` 分支已验证代码**
  - `src/slam/` — SLAM Toolbox + rtabmap 建图
  - `src/navigation/` — Nav2 自主导航 (TEB/DWB)
  - `src/jetauto_peripherals/` — 舵机控制、键盘遥控、Astra Pro 驱动
  - `src/interfaces/` — 坐标/位姿消息定义
  - `src/ros_robot_controller_msgs/` — 与 Pi3 通信的控制板消息
  - 启动脚本: `jetson_start.sh`, `jetson_teleop.sh`, `jetson_slam.sh`, `jetson_navigation.sh`, `jetson_rviz.sh`

### 清理
- 删除 `slam/test/`、`navigation/test/` 共 6 个模板测试文件
- 删除 `interfaces/` 中 11 个未编译的 ROS1 遗留消息/srv
- 删除 `interfaces/mapping_rules.yaml` (ROS1→ROS2 映射，已无用)
- 删除 `interfaces/CMakeLists.txt` 中对应的 `install(FILES mapping_rules.yaml ...)` 行
- 删除 `jetauto_peripherals/setup.py` 中 2 个死 entry point (`joystick_control`, `tf_broadcaster_imu`)
- 删除 5 个纯 rviz 启动文件 (rviz_slam, rviz_rtabmap, rviz_navigation, rviz_rtabmap_navigation, rtabmapviz)
- 删除 6 个无人引用的 rviz 配置文件
- 删除所有 `__pycache__` 目录

### 仓库重组
- HiWonder_JetAuto 仓库 `jetson/` 目录已移除，该仓库现仅保留 Pi3 底层控制代码
- HiJetson 仓库 `lite` 分支作为已验证的 HiWonder Jetson 代码存档 (只读)
- `main` 分支成为完整的 Jetson 机器人工作空间 (感知 + 控制)
