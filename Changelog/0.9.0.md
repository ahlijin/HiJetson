# 0.9.0 (2026-05-31)

## 项目状态

HiJetson — Jetson Orin Nano × Orbbec Astra Pro 多模态感知系统。语音采集+识别流水线已跑通，视觉模块框架搭建完成待开发。

## 已完成

### 语音模块 (Voice Pipeline)

- [x] **音频采集 (voice_capture)**
  - sounddevice 驱动，16kHz/2ch → 自动混音单声道
  - 支持 ALSA 直通 (device=0) 和 PulseAudio (device=-1)
  - 多通道自动混音

- [x] **语音活动检测 (voice_vad)**
  - WebRTC VAD (webrtcvad 2.0.10)
  - 支持 4 级敏感度配置 (vad_mode 0-3)
  - 静音超时自动切分语音片段

- [x] **语音识别 (voice_asr)**
  - 引擎: openai-whisper (PyTorch CUDA)
  - 模型: tiny/base/small 可选，默认 tiny
  - GPU 推理约 0.3s (tiny) / 0.5s (base)
  - 发布 /voice/asr_result (String) 和 /voice/voice_command (VoiceCommand)

## 已知问题

1. **PulseAudio 冲突** — Jetson 上 PulseAudio 默认占用 USB 音频设备，需启动前 `systemctl --user stop pulseaudio.service pulseaudio.socket`
2. **numba + coverage 不兼容** — openai-whisper 依赖 numba 0.64/0.65，需 `pip install "coverage==6.5.0"`
3. **VAD 环境噪声** — 风扇噪声被 VAD 误判为语音，导致产生大量短片段，需调高 vad_mode 或添加噪声门限
4. **astream_camera 驱动** — Orbbec 相机驱动已作为 submodule 集成但未在实际硬件上验证
5. **视觉节点** — image_preprocess、object_detection、depth_processor 代码框架已创建但未开发
